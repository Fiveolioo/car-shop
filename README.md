# Car Shop

The app creates and allows users to set up appointments for their cars. 

### Installing

Clone the repo and install dependencies

```bash
$ git clone https://github.com/Fiveolioo/car-shop.git
$ bundle install
```

Migrate DB:SEED

```bash
$ rake db:migrate
$ rake db:seed
```

## Running the tests

```bash
$ rails s
```
to launch server.   
The app should be running `localhost:3000`
